# Smart_ETL
Upload any messy CSV/Excel → Get perfectly cleaned file in seconds.
Smart Data Cleaner – Automated ETL Web Application  
• Built a production-ready data cleaning tool using Streamlit & Pandas that removes duplicates and intelligently fills missing values  
• Implemented in-memory processing with BytesIO for instant downloads 

# Features
- Removes duplicates automatically  
- Fills missing values (mean for numbers, mode for text)  
- Real-time feedback with green/red status  
- Instant Excel download (in-memory, no server files)  
- 100% works on Streamlit Cloud (zero config)

# Tech Stack
Streamlit • Pandas • Python
